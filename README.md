# Faithless 16x Permanent Texturepack with Multiplayer Support

-This Texturepack REPLACES the "res" folder in "\Common\"!!!.

-Please backup your original "res" folder or rename it to "res backup".

-This texturepack is only tested on LCE Revelations.

-To use this Texturepack Download the latest release and put the "res" folder inside your "\Common\" folder.
